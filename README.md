# Customer Churn Prediction for Gyms and Coaching Centers

## Project Overview

This project predicts customer churn using machine learning techniques. It analyzes customer behavior and demographic data to identify whether a customer is likely to continue or leave a service such as a gym or coaching center.

## Objective

The goal of this project is to help businesses identify customers at risk of leaving so that they can take preventive actions and improve customer retention.

## Dataset

The project uses the Gym Customers Features and Churn dataset from Kaggle.
It includes:

* Membership duration
* Attendance frequency
* Payment behavior
* Customer demographics

Target variable:

* Churn (0 = No, 1 = Yes)

## Models Used

* Logistic Regression (baseline model)
* Random Forest (ensemble model)
* XGBoost (boosting model)

## How It Works

* Data preprocessing and cleaning
* Feature selection and transformation
* Model training using multiple algorithms
* Model comparison and evaluation
* Prediction of customer churn

## Output

The system predicts:

* Customer will stay
* Customer will churn

## Technologies Used

Python
Scikit-learn
XGBoost
Pandas and NumPy

## Conclusion

Among the models used, XGBoost performs best in predicting customer churn. The system can help businesses take early actions to reduce customer loss and improve retention strategies.
